# 🧮 WebGPU Compute & Alignment

Deep dive into WebGPU compute passes and manual memory management.

## 🚀 Key Features
- **Compute Shaders**: Procedural data processing on the GPU.
- **Memory Alignment**: Handling data layout and offsets manually.
- **Buffer Management**: Optimized data transfer between CPU and GPU.

## 🏃 Running Locally
\`\`\`bash
# From the root directory
bun dev --filter webgpu-compute
