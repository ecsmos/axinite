# 🎬 Three.js WebGPU Demo

Exploring WebGPU integration within the **Three.js** ecosystem.

- **Three.js WebGPU**: Using the modern Three.js WebGPU renderer.
- **Node-based Materials**: Leveraging the new node system for shaders.
- **Real-time Graphics**: Interactive high-performance rendering.
