# 🔺 WebGPU Clean Triangle

The "Hello World" of WebGPU. A basic, high-performance renderer starting point.

## 🚀 Key Features
- **Modern Renderer**: Basic WebGPU setup.
- **Vertex & Fragment Shaders**: Fundamental WGSL shader implementation.
- **Clean Foundation**: Minimalistic code for learning WebGPU.

## 🏃 Running Locally
\`\`\`bash
# From the root directory
bun dev --filter webgpu-triangle
