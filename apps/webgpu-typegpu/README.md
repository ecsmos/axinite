# ⚡ WebGPU with TypeGPU

Exploring type-safe GPU programming with the **TypeGPU** library.

## 🚀 Key Features
- **Type-safe Shaders**: Building GPU logic using TypeScript.
- **Auto Memory Layouts**: TypeGPU automatically generates memory schemas.
- **Simplified Workflow**: Higher-level abstraction over raw WebGPU.

## 🏃 Running Locally
\`\`\`bash
# From the root directory
bun dev --filter webgpu-typegpu
