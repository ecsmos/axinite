# 📦 @axinite/assets

A browser-side asset loading library for high-performance WebGPU applications.

## 🚀 Features
- **Worker Loading**: Offloads asset fetching and decoding to a separate thread.
- **Comlink Support**: Simplified communication between main and worker threads.
- **Optimized Formats**: Ready to handle textures, meshes, and more.

## 🛠️ Usage
Used across Axinite apps for data-oriented asset loading.
